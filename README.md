## Project Description
SmartScript-DevOps is a beginner-friendly DevOps project demonstrating Git collaboration, CI/CD with GitHub Actions and cloud deployment using a simple HTML & JavaScript application.


## Live Deployment
https://smartscript-devops.netlify.app/


## Technologies Used
- HTML5, CSS3, JavaScript
- QRCode.js
- GitHub Actions
- Netlify

## Features
- SLMC-based login (mock validation)
- Digital prescription creation
- QR code generation
- Save QR code as image
- Responsive UI
